EntityEncoder

Swift

public protocol EntityEncoder

A strategy for transforming Entity instances to and from raw byte data.

Provided primarily to help EntityCache implementations.

  • encodeEntity(_:)

    Returns entity, including all of its metadata, serialized as raw data. Returns nil if this encoder does not know how to encode the entity.
